# 操作系统小组实验
**目的与技术**
- 暂定使用O(1)调度算法和银行家算法；
- 各个算法模块分开以便分工共同开发。

**数据结构**
- 具体请查看代码注释。

**关于test**
- 测试数据使用的是自定义数据，并在运行之后再模拟插入一个数据，通过银行家算法检测；
- test的数据可控，能够可视化即可。

**下载说明**
- git clone或者直接下载；
- 下载后用vs2017打开.sln项目文件即可；
- Myhead.h是唯一头文件，更新头或结构体请及时推送。
